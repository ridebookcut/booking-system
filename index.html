🚀 Muốn chuyển sang Supabase? (Free plan hoàn toàn khả thi!)

  1. Tạo project tại supabase.com (free: 500MB DB, 2GB bandwidth)
  2. Tạo table bookings với cột: id, date, time, customer_name, customer_phone, service_request, status, timestamp
  3. Tạo table days_off với cột: date
  4. Lấy SUPABASE_URL và SUPABASE_ANON_KEY từ Settings → API
  5. Thay thế Firebase SDK bằng: import { createClient } from '@supabase/supabase-js'
  6. Đổi db.collection().get() → supabase.from('bookings').select('*')
  7. Auth: dùng supabase.auth.signInWithPassword() thay auth.signInWithEmailAndPassword()
